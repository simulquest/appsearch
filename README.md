# APPSearch 🚀

**APPSearch** est un lanceur d'applications ultra-rapide et moderne pour Windows, développé en **Rust** avec l'interface **Slint UI**. Il est conçu pour remplacer le menu Démarrer lent et encombré par une expérience fluide, élégante (Glassmorphism) et axée sur la productivité.

## ✨ Fonctionnalités

- ⚡ **Recherche Instantanée** : Indexation ultra-rapide des applications, fichiers et dossiers.
- 🧮 **Outils Intégrés** : Calculatrice, convertisseur d'unités et de devises, sélecteur de couleurs Hex/RGB.
- 🌐 **Modes Web** : Recherchez directement sur YouTube, Spotify ou Google en utilisant `TAB`.
- ⏰ **Productivité** : Lancez des timers et des rappels instantanément (ex: `timer 5m`).
- 📂 **Multi-lecteurs** : Support complet des disques secondaires et détection automatique des clés USB.
- 🦀 **Propulsé par Rust** : Performance maximale, sécurité mémoire et consommation de ressources minimale (< 50 Mo de RAM).

## 🚀 Démarrage Rapide

### Raccourci par défaut
Appuyez sur `Ctrl + Shift + K` pour ouvrir la barre de recherche de n'importe où sur votre PC.

### Installation
Téléchargez la dernière version depuis l'onglet [Releases](https://github.com/simulquest/APPSearch/releases) ou utilisez notre [site web](https://appsearch.rf.gd).

## 🛠️ Installation pour les développeurs

### Prérequis
- [Rust](https://www.rust-lang.org/tools/install) (dernière version stable)
- [Slint](https://slint.dev/) (les dépendances seront gérées par Cargo)

### Build
```bash
git clone https://github.com/simulquest/appsearch.git
cd appsearch
cargo build --release
```
